This project is intended solely for interoperability between lawfully obtained
NWN:EE clients and authorized legacy NWN 1.69 servers.

Any reverse engineering was limited to identifying functional interface
information necessary for compatibility, including packet formats, protocol
state, and wire-level behavior. The implementation is independent and does not
include proprietary game code, server code, client binaries, copyrighted assets,
or CD keys.

The project must not be used to bypass ownership checks, share authentication
material, evade server authorization, or distribute proprietary game content.
